# Organização
A organização do projeto em termos de arquivos segue a organização "padrão" de um projeto em Angular. A estrutura de pastas é a seguinte:

    SPA_wiki
    ├───e2e
    ├───node_modules
    ├───src
    │   ├───app
    │   │   ├───components
    │   │   │   ├───cada componente 
    │   │   ├───guards
    │   │   ├───models
    │   │   ├───services
    │   │   └───shared
    │   ├───assets
    │   ├───environments
    │   └───styles

Estando a pasta de wiki no seu diretório próprio.
