## Introduction
Este caso de uso descreve a funcionalidade de cancelar uma conta anonimizando os dados pessoais que tenham sido recolhidos. Esta funcionalidade é uma extensão da funcionalidade de [criar uma conta de utilizador](UC1.a.md).
Este documento descreve o processo de **anonimização** e dos dados pessoais de um utilizador e do seu **soft delete**.
