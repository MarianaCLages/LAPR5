Projeto SEM_PI_S5

Módulo SPA (Single Page Application), também conhecida como "FrontEnd".
Esta aplicação vai ser responsável por interagir com os vários utilizadores conhecidos no negócio.

O domínio contempla as Entidade do negócio, os Serviços (ou casos de uso) envolvendo essas entidades e DTOs (in/out para os serviços de dominio).
